# stock

这里存放个股分析 Markdown 文档。

## 目录规则

- 个股分析统一存放在 `stock/`
- 不在工作区根目录存放个股分析副本
- 文件名统一使用 `股票名-YYYY-MM-DD.md`

## 当前文档

- `和邦生物-2026-04-21.md`
- `金发科技-2026-04-21.md`
- `隆平高科-2026-04-23.md`

## 推荐写法

每篇分析建议包含以下部分：

1. 核心结论
2. 关键财报数据
3. 买点逻辑
4. 卖点与风险点
5. 波段交易建议
6. 简版执行卡片
7. 一句话判断

后续新增个股分析时，继续按同样格式补充即可。
